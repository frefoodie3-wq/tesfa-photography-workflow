# Tesfa Photography Client Proofing

A static client gallery workflow for Tesfa Photography.

The site supports the basic professional proofing flow:

1. Create a gallery.
2. Upload preview images.
3. Send a private client link.
4. Let the client select favorites.
5. Receive the selected filename list for editing and later delivery.

Open `index.html` directly or serve the folder with any static host.
